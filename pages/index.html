  • This website is running on a NodeJS server.
  • The server handles requests through url and sends responses in HTML or CSS
  • Every different url requests a different page and the server routes the request to the correct document.
  • The document then requests any other files it may need, like script files or stylesheets.
  • If no file can be found matching the requested url a 404 error is thrown.
  • Instead of this crashing the server, it routes the bad request to a custom 404 page.
